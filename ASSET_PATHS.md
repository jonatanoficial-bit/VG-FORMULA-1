## Regra fixa para próximos upgrades
1. Sempre consultar este arquivo antes de alterar `index.html`, `styles.css` e `app.js`.
2. Sempre gerar ZIP completo do jogo sem incluir os assets pesados.
3. Sempre manter versão, data e hora visíveis no jogo.
4. Sempre priorizar mobile-first.
