VG Motorsport Manager - Build v0.1.5

Como usar:
1) Extraia este ZIP em uma pasta.
2) Extraia os ZIPs de assets na mesma pasta, respeitando os caminhos oficiais.
3) Abra index.html.

Regra do projeto:
- cada upgrade gera ZIP completo do jogo, mas sem os assets pesados.
- antes de qualquer alteração, consulte ASSET_PATHS.md e DEVELOPMENT_PLAN.md.
- versão, data e hora ficam visíveis dentro do jogo.
